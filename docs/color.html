# Color 色彩

统一的色彩搭配可以提高品牌的识别度，色彩的运用除了需要考虑品牌的统一性之外，还需要达到信息传递，交互反馈等目的。Taro UI 是凹凸实验室出品，品牌颜色为「凹凸蓝」，因此 Taro UI 的整体配色风格也是基于「凹凸蓝」展开的。
